Python — это высокоуровневый язык программирования общего назначения. Он был создан Гвидо ван Россумом и впервые выпущен в 1991 году.

Основные особенности Python:
- простота синтаксиса;
- динамическая типизация;
- интерпретируемость;
- поддержка ООП и функционального программирования.

В 2020 году Python стал одним из самых популярных языков программирования в мире.

Модули Python позволяют организовывать код в отдельные файлы и упрощают повторное использование.

Функция print() используется для вывода данных на экран.
